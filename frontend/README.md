# frontend

### Running the code

```
First, ensure you are in the right directory.
cd .\frontend

Then you can run:
npm install

Then run:
npm run serve

You should see confirmation everything is running in the terminal.

If everything is running successfully, you will see:

App running at:
- Local: http://localhost:porthere
- Network: http://ipaddress:port


Navigate to the localhost link that is displayed in the terminal using a web browser.
```
